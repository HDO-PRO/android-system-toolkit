# 📱 Android Tools & System Utilities

A curated collection of useful Android tools, system utilities, DNS tools, VPN apps, and privacy-focused applications.

> ⚠️ Note: This list is for educational and personal use. Apps and tools are frequently updated, so features and availability may change over time.

---

## 🧰 System Tools

Tools for device optimization, automation, and system control.

- **[Termux](https://termux.dev)** - Powerful terminal environment for scripting, package management, and running Linux tools on Android
- **[SD Maid](https://sdmaid.app)** - Cleans leftover files, cache, and unnecessary system junk to free up storage
- **[Shizuku](https://shizuku.rikka.app)** - Allows apps to use system APIs directly with adb/root privileges without root
- **[LSPosed](https://lsposed.org)** - Framework for running modules without modifying system partitions (requires root)
- **[Magisk](https://github.com/topjohnwu/Magisk)** - Systemless root solution for Android devices
- **[KernelSU](https://kernelsu.org)** - Root solution for Android kernel

---

## 🌐 DNS & Privacy Tools

Improve speed, privacy, and security with custom DNS solutions.

- **[1.1.1.1 (Cloudflare WARP)](https://one.one.one.one)** - Fast, privacy-focused DNS with optional VPN-like protection
- **[NextDNS](https://nextdns.io)** - Fully customizable DNS with ad-blocking, parental controls, analytics, and security filtering
- **[AdGuard](https://adguard.com)** - Blocks ads, trackers, and malicious domains at the DNS level across apps and devices
- **[RethinkDNS](https://rethinkdns.com)** - Open-source DNS resolver with firewall capabilities
- **[ControlD](https://controld.com)** - Privacy-focused DNS with customizable filtering

---

## 🔐 VPN Tools & Privacy

Secure your connection and protect your online identity.

- **[Proton VPN](https://protonvpn.com)** - Privacy-first VPN with strong encryption and strict no-logs policy
- **[WireGuard](https://www.wireguard.com)** - Lightweight, fast, and modern VPN protocol known for performance
- **[OpenVPN](https://openvpn.net)** - Widely used open-source VPN solution for secure encrypted connections
- **[Mullvad VPN](https://mullvad.net)** - Privacy-focused VPN with anonymous account system
- **[IVPN](https://ivpn.net)** - No-logs VPN with multi-hop connections

---

## 🔒 Firewall & Network Security

Control app network access and enhance security.

- **[NetGuard](https://netguard.me)** - No-root firewall for blocking internet access per app
- **[AFWall+](https://github.com/ukanth/afwall)** - Firewall for rooted Android devices
- **[RethinkDNS + Firewall](https://rethinkdns.com)** - DNS resolver with built-in firewall
- **[Blokada](https://blokada.org)** - Ad blocker with firewall capabilities

---

## ⚙️ Automation & Productivity

Extra tools that improve usability and device control.

- **[Tasker](https://tasker.joaoapps.com)** - Powerful automation tool for creating custom triggers and actions
- **[KDE Connect](https://kdeconnect.kde.org)** - Sync phone and PC for file sharing, notifications, clipboard, and remote control
- **[Automate](https://llamalab.com/automate)** - Automation app with flowchart-based programming
- **[MacroDroid](https://macrodroid.com)** - Simple automation tool for Android

Tools for managing files and storage.
